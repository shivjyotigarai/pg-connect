Welcome to PG Connect API !!!

Welcome to version 1.0 of The PG Connect Database API , a public service API all about fictional rooms , users , owners , reviews etc free to use when making your fancy new App, Website or Service. Below you will find a current list of the available options .

  • Get all fictional rooms present on our database along with all the details
  • Filter/Sort rooms based on various parameters such price , review , ratings etc
  • Get the details of a specific room
  • Register yourself as a user/owner
  • Add/Update/Delete rooms if you registered as an owner
  • Write reviews and give ratings for the rooms
  • Reset your password if you forget your login credentials

Variables Values
{{URL}} https://pg-connect.herokuapp.com
{{roomId}} Id of the room that you want to search/delete/update
{{jwt_owner}} JSON Web token that you get once you are logged in
{{userId}} Id of the user that you want to search/delete/update
{{user_email}} Registered email account of the user
{{owner_email}} Registered email account of the owner
{{reset_password_token}} Reset password token that you get via mail once you forget your credentials
